Converts a PEAR\_\-LOG\_\-$\ast$ constant into a syslog LOG\_\-$\ast$ constant.

This function exists because, under Windows, not all of the LOG\_\-$\ast$ constants have unique values. Instead, the PEAR\_\-LOG\_\-$\ast$ were introduced for global use, with the conversion to the LOG\_\-$\ast$ constants kept local to to the syslog driver.
